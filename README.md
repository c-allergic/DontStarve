# DontStarve
## 🎮 游戏简介：饥荒·黄金时代

**《饥荒：黄金时代》** 是一款网页版的生存小游戏，你将扮演一名迷失荒野的幸存者，在随机生成的无限世界里，靠采集、狩猎、建造与战斗活下去，目标只有一个：**活得更久一点，再更久一点。**

- **世界特点**：  
  - 🌿 草原：资源均衡，适合建家，有浆果、兔子、鱼塘  
  - 🌲 森林：树多、资源偏木材  
  - 💀 荒地：怪物多、资源少，高风险高回报  
- **核心循环**：  
  白天探索采集 → 黄昏准备防御 → 夜晚围着营火苟住 → 逐渐建起自己的小基地。

---

## 🎯 基本操作与快捷键

- **移动**：`W A S D` 或 方向键  
- **交互 / 攻击**：**鼠标左键** 点击附近目标  
- **远程射击（弓箭）**：**鼠标右键** 指向要射击的位置  
- **背包**：`E` 打开 / 关闭  
- **制作面板**：`H` 打开 / 关闭  
- **成就面板**：`T` 打开 / 关闭  
- **保存游戏**：界面右下角点击 **「保存」**  
- **重置存档**：点击 **「重置」**（会清空进度）  
- **音乐开关**：点击 **「音乐」** 按钮

---

## ❤️ 生存三大核心属性

- **生命值（❤️）**  
  - 归零即死亡，注意战斗和饥饿、天气等间接伤害  
  - 靠吃好东西、营火附近缓慢恢复、睡觉来回满

- **饱食度（🍗）**  
  - 不吃东西会逐渐掉血  
  - 浆果、肉类、烤肉都能回复饱食度，效果不同

- **理智值（🧠）**  
  - 低理智会削弱你的战斗力，甚至出现更多威胁  
  - 吃好食物、远离恐怖环境、稳定生活都有帮助

## 🧺 背包与资源

打开 `E` 可查看主要资源：

- **基础材料**：树枝、干草、燧石、木材、石头、金块……
- **高级材料**：绳索、编织布、羊毛、羊油等（多通过制作或击杀获得）
- **食物**：浆果、小肉、大肉、腐肉等  
- **种植**：松果可种树，点击 `种植松果` 按钮即可在脚下种树 🌱

建议前期优先采集：

- 🌿 干草、树枝、燧石：解锁基础工具  
- 🌳 木材、石头：营火、床、防御塔等建筑  
- 🍇 浆果：早期最稳妥的食物来源

---

## 🛠 制作与建筑（H）

打开 `H` 制作面板，可以打造关键工具与建筑：

- **工具类**  
  - 🪓 斧头：砍树更快  
  - ⛏️ 矿镐：挖石头更快  
  - 🗡️ 长矛：近战主力武器  
  - 🏹 弓箭：远程攻击，需要制作箭矢  
  - 🎣 鱼竿：在鱼塘钓鱼用的（很重要的食物来源）

- **建筑类**  
  - 🔥 营火：夜间保命核心，附近还能缓慢回血  
  - 🛏️ 床：消耗大量饱食度，快速跳到第二天并回复状态  
  - 🗼 灯塔：远方的导航标记，方便找到自己的家  
  - 🛡️ 防御塔：自动攻击附近敌人，守家神器

- **进阶生存**  
  - ➰ 绳索 → 🧱 编织布 → 👕 护甲、🛡️ 挡风板等  
  装备护甲可以大幅减伤，让你更敢打。

## 🐟 鱼塘与钓鱼指南

- **鱼塘刷新**  
  - 只在 **草原** 区域生成（绿色地带）  
  - 现在鱼塘的刷新概率已大幅提高：**在草原多走几个区块基本都能遇到鱼塘**

- **如何开始钓鱼**  
  1. 在制作面板里做出 **鱼竿（🎣）**  
  2. 走到鱼塘边缘，**鼠标左键** 点击鱼塘  
  3. 日志会提示「开始钓鱼... (需要4秒)」  
  4. 4 秒内保持在鱼塘附近，等待结果

- **钓鱼注意点**  
  - 距离鱼塘太远会中断钓鱼  
  - 鱼竿有耐久，用完会损坏，需要重新制作  
  - 成功时获得小肉（🍖），是非常稳定的肉类来源

## 🌙 昼夜与危险

- **白天**：适合探索、采集、制作、建家  
- **黄昏 / 夜晚**：  
  - 怪物更危险，视野变差  
  - 一定要回到营火附近，保持明亮  
  - 夜里更容易遇到夜行怪，建议站在营火和防御塔附近
